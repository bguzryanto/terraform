# `file_exists` Function

`file_exists` determines whether a file exists at a given path

```hcl
file_exists(path)
```

## Examples

```
> file_exists("${path.module}/hello.txt")
true
```

```hcl
file_exists("custom-section.sh") ? file("custom-section.sh") : local.default_content
```
